# 🚨 Fraud Detection Messenger

Веб-приложение (мессенджер) с возможностью интеллектуального анализа сообщений для выявления мошеннической активности.

Проект реализует клиент-серверную архитектуру и включает систему анализа сообщений на основе гибридного подхода: rule-based фильтрации и модели машинного обучения.

---

## 📌 Основные возможности

- 💬 Обмен сообщениями в режиме реального времени  
- ⚡ Поддержка WebSocket для мгновенной доставки сообщений  
- 🧠 Анализ сообщений на наличие признаков мошенничества  
- 🔐 Авторизация пользователей  
- 📊 Подготовка к интеграции модели машинного обучения  

---

## 🏗️ Архитектура проекта

Проект разделён на несколько основных компонентов:

- **Frontend** — одностраничное приложение на Vue.js  
- **Backend** — сервер на Django + Django REST Framework  
- **Real-time слой** — Django Channels (WebSocket)  
- **ML модуль** — классификация сообщений (DistilBERT)  
- **Контейнеризация** — Docker

## 🧠 Модуль анализа сообщений

Система использует гибридный подход:

1. **Rule-based анализ**
   - ключевые слова  
   - регулярные выражения  
   - структурные признаки  

2. **Модель DistilBERT**
   - бинарная классификация сообщений  
   - оценка вероятности мошенничества  

---

## 🎤 Обработка голосовых сообщений

Голосовые сообщения обрабатываются следующим образом:

1. Распознавание речи (Speech-to-Text)  
2. Получение текстовой транскрипции  
3. Передача текста в модуль анализа
